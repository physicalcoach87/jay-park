📌 무슨 시스템인가

프로 축구팀 피지컬 코치가 직접 기획·의뢰하여 개발한 선수 관리 전용 SaaS입니다.
Catapult GPS 데이터를 업로드하면 운동부하·부상위험을 자동 분석하고, 선수에게 개인 맞춤 운동 처방을 발송합니다. 추후 타 팀 납품 또는 SaaS 전환을 목표로 멀티테넌트 구조로 설계되어 있습니다.

초대 링크 가입 방식

이메일을 미리 알 필요 없이 초대 링크 하나로 가입부터 권한 설정까지 완료됩니다.

슈퍼어드민 → [🔗 초대] 클릭
→ 팀 + 역할 선택
→ UUID 토큰 생성 (유효 7일, 1회용, invite_tokens 테이블)
→ https://.../?invite={token} 카카오로 전달

신규 유저 → 링크 클릭
→ 이름 + 이메일 + 비밀번호만 입력
→ club_id + role 자동 세팅 (승인 불필요)
→ 바로 로그인 가능

⚙️ 운동 모드 3가지

선수 앱 보급 여부, 운영 방식에 따라 팀별로 선택. clubs 테이블의 3개 플래그로 제어.

📱 풀 모드
선수들이 직접 앱으로 웰니스(컨디션)와 RPE(운동 강도)를 입력. 코치는 확인·수정만 담당.
use_player_app: true · use_rpe: true · use_wellness: true
✏️ 코치 RPE 모드
선수 앱 없이 코치가 GPS 엑셀 파일에 RPE·체중 컬럼을 추가해서 업로드. 업로드 시 자동으로 session_records, wellness_records에 저장됨.
use_player_app: false · use_rpe: true · use_wellness: false
📡 GPS 전용
GPS 데이터만 사용. RPE·웰니스 기능 전부 비활성화. 단순 부하 모니터링만 필요한 팀.

sRPE · ACWR 계산

// 세션 운동부하
sRPE = rpe × duration  // 예: RPE 7 × 75분 = 525 AU

// 운동부하 급성/만성 비율 (ACWR)
acute   = 최근 7일 sRPE 합산
chronic = 최근 28일 sRPE 평균 × 7
acwr    = acute / chronic

// 판정
acwr > 1.5  → 🔴 고위험 (부상 위험 급증)
1.3 ~ 1.5   → 🟡 주의
0.8 ~ 1.3   → 🟢 안전 구간
< 0.8      → 🟡 과소부하 (컨디션 저하 가능)
